# Changelog

## [0.1.0] - 2026-02-21

### Added
- Initial release of the PHP Composer package `mockeroo/mock-responses`.
- `MockResponses::getResponse(int $statusCode)` — returns a random sarcastic response or null.
- `MockResponses::getAvailableCodes()` — returns all supported HTTP status codes.
- Requires PHP >= 8.1.
